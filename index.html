JavaScript Flashcards

JavaScript (JS)

JavaScript is the programming language of the internet.

Operators

The basic JavaScript operators are + - * / %.

Variables

Variables are used to store numbers, strings, characters, and booleans.

Variable Naming Conventions

Variables are typically named in camel case, eg, camelCaseVariable.

Functions

A JavaScript function is a block of code that can be invoked when needed, as many times as needed.

Methods

In JavaScript, a method is a property containing a function definition.

Arguments

Arguments are the information passed to functions.

Parameters

Parameters are the assigned names to the arguments passed to functions.

Return

In JavaScript, a function can return a value when called.

Chaining Methods

Multiple methods can be chained and nested together to perform complex actions.

Strings

A string variable is used to hold a series of characters, such as a word or a sentence.

Booleans

Booleans are variables that can hold only the values "true" and "false".

Undefined

A variable is "undefined" when it has not yet been assigned a value.

NaN

NaN is a value that a numeric variable can hold when passed a value that is not a number.

Escape

An escape character is used to insert special characters into a string.

Alert

Alert is used to create a popup window containing text.

Comments

Comments are used to add description to code to help in understanding it.

jQuery

A library of JavaScript functions.

Attributes

A description for an HTML element.
